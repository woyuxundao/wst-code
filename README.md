# wst-code

#### 介绍
  使用typescript 语言编写网页，用于识别固定规则的barcode的正确与否。使用fetch 获取json数据。
  短期：人工修改json文件， 长期使用web方式修改
#### 软件架构
  学习typescript 结合公司内部需求制作


#### 安装教程

1.  npm run build
2.  把文件html ，js ，css，assert文件放入web服务中


#### 使用说明

1.  局域网登录服务器即可，使用扫描枪扫描即可
